# Fabelio Product Scrapper

This app just scrape product name, slug, price, description and price from detail product page at [Fabelio.com](https://fabelio.com/).

## Instalation
Install node package with:
```bash
yarn install
#or
npm install
```

## Usage
To running application:
```bash
# For development
yarn dev

# For production
yarn start

#For test
yarn test
```
